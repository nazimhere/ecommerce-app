# Velora - Full-Stack E-Commerce Platform 🚀

**Velora** is a modern, responsive e-commerce website built with the **MERN stack** (MongoDB, Express.js, React, Node.js). This project demonstrates full-stack web development skills including user authentication, product management, shopping cart functionality, and secure payment integration.

## ✨ **Key Features**

- ✅ **User Authentication** (JWT/Session-based login/register)
- ✅ **Product Catalog** with Search & Filters  
- ✅ **Shopping Cart** & **Wishlist**
- ✅ **Admin Dashboard** (CRUD operations)
- ✅ **Order Management** & Tracking
- ✅ **Responsive Design** (Mobile/Tablet/Desktop)
- ✅ **Payment Integration** Ready (Stripe/Razorpay)
- ✅ **Deployed on Render** with Custom Domain

## 🛠 **Tech Stack**

| **Frontend** | **Backend** | **Database** | **Deployment** | **Tools** |
|--------------|-------------|--------------|----------------|-----------|
| React 18+ | Node.js/Express | MongoDB/Mongoose | Render | Axios/Postman |
| Tailwind CSS | JWT Auth | Cloudinary (Images) | Custom Domain | Thunder Client |
| React Router | RESTful APIs | Environment Vars | HTTPS/SSL | Git/GitHub |
